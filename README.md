# TCC API

Este repositório contém a API desenvolvida com FastAPI para o nosso Trabalho de Conclusão de Curso (TCC). Ela será responsável por gerenciar e fornecer os dados utilizados no sistema que estamos construindo.

### Tecnologias

- Python 3.11+

- FastAPI

- Uvicorn

- SQLAlchemy / ORM

### Status do projeto

Em desenvolvimento
